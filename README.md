# NOIR ATELIER

Premium clothing brand business website built for brand storytelling, collection showcase, and store/contact conversion.

This project is a multi-page fashion website created with React and Vite. It is designed as a premium editorial-style brand presence rather than a full e-commerce storefront.

## Overview

NOIR ATELIER is positioned as a modern clothing branding studio with a high-end visual identity. The site focuses on:

- brand promotion
- collection showcase
- editorial lookbook presentation
- trust-building business content
- a light shop-style listing page for product discovery

## Features

- cinematic intro reveal for the `NOIR ATELIER` brand
- responsive multi-page layout
- premium hero slider and motion-led storytelling
- collection and lookbook sections with curated imagery
- services, testimonials, about, and contact pages
- shop page with product cards, pricing, and product details
- subtle animated background and Framer Motion transitions

## Pages

- `/` - Home
- `/collections` - Collections
- `/lookbook` - Lookbook
- `/about` - About
- `/services` - Branding Solutions / Services
- `/testimonials` - Testimonials
- `/contact` - Contact
- `/shop` - Product Listing

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Then open:

[http://localhost:5173](http://localhost:5173)

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Deployment

This project is ready to deploy on any static-friendly frontend host.

### Vercel

1. Import the repository into Vercel.
2. Framework preset: `Vite`
3. Build command:

```bash
npm run build
```

4. Output directory:

```bash
dist
```

### Netlify

Use:

- Build command: `npm run build`
- Publish directory: `dist`

## Design Notes

The visual system uses:

- warm cream and beige backgrounds
- deep brown / near-black contrast sections
- copper accent highlights
- restrained typography with strong editorial hierarchy
- subtle motion intended to support, not overpower, the content
